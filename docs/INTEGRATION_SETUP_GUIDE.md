## Step-by-Step Setup for Each Integration

### 1Ô∏è‚É£ **Google AI (Gemini)** - AI Content Generation

**What it does:** Generate email copy, SMS messages, marketing content automatically.

**Steps:**
1. Go to: https://aistudio.google.com/app/apikey

### 2Ô∏è‚É£ **Cal.com** - Appointment Booking

**What it does:** Customers book appointments, you get reminders via Email/SMS/WhatsApp.

**Steps:**
1. Go to: https://app.cal.com/settings/developer/api-keys
2. Click **"Add"** button
3. Name it: `OmniFlow`
4. Set Expiration to: **"Never expires"**
5. Click **"Save"**
6. **Copy the key** (starts with `cal_...`) - shown only once!
7. Paste in **OmniFlow ‚Üí Settings ‚Üí Cal.com ‚Üí API Key field**
‚úÖ Done!

### 3Ô∏è‚É£ **Brevo** - Email Marketing

**Steps:**

6. In **OmniFlow Settings ‚Üí Brevo**, fill:
   - **API Key**: Paste here
   - **Sender Email**: your@business.email
   - **Sender Name**: Your Business Name
7. Click **"Save API Keys"**

## üÜò Troubleshooting

### "Failed to save API key"
- ‚úÖ Make sure you copied the ENTIRE key (sometimes it gets cut off)
- ‚úÖ Try again slowly
- ‚úÖ Check for extra spaces at beginning or end of key

### "Authentication failed" error
- ‚úÖ Verify key is active in the original service
- ‚úÖ Some services need you to enable the key after creating it
- ‚úÖ Generate a new key if old one isn't working

### "Key expired" error after 30 days
- ‚úÖ Some services expire keys
- ‚úÖ Generate new key and update in OmniFlow

## üí° Best Practices

‚úÖ **Keep API keys safe:**
- Never share them via email
- Never paste them in public documents
- Never commit to GitHub

‚úÖ **Use long-lasting keys:**
- Set expiration to "Never" when possible
- Keep production keys separate from test keys

‚úÖ **If key is compromised:**
- Regenerate immediately
- Delete old key
