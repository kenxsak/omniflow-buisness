# **App Name**: OmniFlow

## Core Features:

- Integrated CRM: CRM and Lead Management: Use Baserow or NocoDB for CRM pipeline and lead tracking. Integrate with AI agents and email via n8n/Pipedream for automated updates.
- Email Automation: Email Marketing: Integrate Brevo API for handling email campaigns and automations. Leverage the free tier for up to 300 emails per day, with options to scale.
- AI Assistant: AI-Powered Assistant: Implement an AI Assistant using Flowise AI to onboard new users, auto-set CRM pipelines, and guide email campaign setups. This tool automates key tasks and enhances user experience.

## Style Guidelines:

- Primary color: Neutral white or light gray for clean interface.
- Secondary color: Soft blue for highlighting interactive elements.
- Accent: Teal (#008080) for key calls to action and important notifications.
- Clear and modern typography for easy readability.
- Simple and intuitive icons to represent different functions and data.
- Clean and organized layout with a focus on usability.
